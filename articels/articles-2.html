Apa itu Software Engineering?


Asa Romi Apr 13, 2019 at 08:00 1 Comments

Pendahuluan - Pengenalan Software dan Software Engineering

Di dalam kehidupan era sekarang ini banyak sekali bidang - bidang yang menggunakan software. Dimana semua perangkat atau peralatan yang digunakan pada berbagai bidang pasti memanfaatkan software di dalamnya. Karena pada dasarnya semua perangkat keras elektronik tidak akan bisa berfungsi tanpa adanya software di dalamnya.

Software adalah program komputer yang isinya prosedur - prosedur mengenai bagaimana pengoperasian sistem komputer.


Software Engineering menurut beberapa Ahli

  1. Pressman, 2014 :
    • Intruksi atau program komputer yang jika dijalankan atau dieksekusi maka akan menjalankan fungsi yang dimiliki sebuah software.
    • Struktur data yang memungkinkan program untuk memanipulasi informasi software itu sendiri secara memadai.
    • Sebuah informasi deskriptif baik dalam bentuk cetak maupun virtual yang menggambarkan operasi dan penggunaan program tersebut.

  2. Sommerville, 2015
    Berisi sebuah program komputer, dan juga berisi dokumentasi dari program tersebut.

Jenis - Jenis Software

  1. Software Market
    Software jenis ini merupakan Software yang disedialam dari kebutuhan pasaran atau user. Dibagi juga menjadi :
    • Software Generik : Perangkat lunak standar yang diproduksi suatu perusahaan pengembang dan dibuat secara umum untuk dijual pada pasar terbuka dan kepada siapapun yang membelinya.
    • Software Pesanan : Perangkat Lunak yang dibuat secara khusus dan disesuaikan dengan permintaan dan kebutuhan khusus dari pelanggan
    Contoh dari Software Market : e-SPT, siakad Perguruan Tinggi, dll

  2. Software Domain Ada beberapa jenis Software Domain diantaranya :
    • System Software : Sekumpulan Program yang ditulis untuk melayani atau menunjang perogram lainnya. Biasanya di dalam System Software ini masih terdapat Software lagi. Contoh : Compilers, Drivers, Sistem Opersai, dll.
    • Application Software : Program mandiri yang mampu menyelesaikan sebuah keperluan tertentu yang dibutuhkan. Perangkat Lunak ini biasanya digunakan untuk mengontrol dan menunjang fungsi kebutuhannya secara real time atau terus menerus.
    • Engineering/Scientific Software : Software yang dicirikan dengan algoritma numerik dan banyak digunakan dalam bidang engineering dan science. Contohnya Computer Aided Design (CAD) yang digunakan untuk mendesain Mobil, System Stimulation pemasaran, dll.
    • Embedded Software : Suatu Software disimpan dalam memori tetap yang digunakan untuk mengontrol produk dan sistem dari Hardwarenya. Software ini dijalankan dengan berbagai fungsi terbatas untuk berkomunikasi dengan intruksi yang dimiliki oleh software yang ada pada suatu Perangkat Keras. Contohnya Remote Control AC, Elevator, dll.
    • Product Line Software : didesain untuk memberikan kemampuan spesifik untuk setiap pelanggaan yang berbeda. Tergantung dengan kebutuhan penggunanya. Contoh : WS dan WP(Word Processing), Lotus dan Supercalc (Spreadsheet), Printshop dan Print Magic (Computer Graphics) serta lain sebagainya.
    • Web Applications : Merupakan program aplikasi yang berbasis web, biasanya dicirikan dengan ditautkan dengan hypertext files. Software ini berkembang menjadi komputasi yang sangat canggih, yang menyediakan untuk fitur mandiri, dan juga terintegrasi dengan database perusahaan dan aplikasi bisnis. Software jenis ini juga tidak terbatas dengan platform, dengan menggunakan sistem operasi apapun.
    • Artificial Intelligence Software : AI Software dibuat dengan menggunakan algoritma non numerik untuk menyelesaikan masalah yang rumita atau masalah yang tidak bisa diselesaikan secara langsung. Contohnya : Robotics, Expert System, Game Playing, dll.

  3. Software Lisensi Ada beberapa jenis Software Domain diantaranya :
    • Open Source Software : Ketika membeli aplikasi maka akan disertakan source codenya. Pengguna akan mendapatkan free source code dan software serta juga bisa memodifikasi software atau source code tersebut. Pengguna dapat menginstall software ke dalam banyak komputer. Ketika ada problem, susah untuk ditangani secara cepat. Contoh : Linux, OpenOffice, GIMP, dll.
    • Proprietary Software : Ketika membeli aplikasi hanya disediakan program hasil dari compile source code saja. Pengguna tidak dapat memodifikasi software. Pengguna juga harus memiliki lisensi dari penjual sebelum menginstallnya ke dalam komputer. Ketika terdapat masalah atau trouble akan mendapat full support dari penjual karena Softwarenya berbayar. Contoh : MS. Windows, MS. Office 2013, Adobe Photoshop, dll.

  4. Open Source Software (Tambahan) Source codenya terbuka dan memiliki hak cipta di dalamnya, namun lisensi yang gratis atau bebas ini memungkinkan pengguna lain untuk menggunakan dan memodifikasi software secara bebas. Berikut ini adalah Jenis Lisensi Software Open Source :
    • GNU General Public License (GPL)
    • Apache License
    • BSD License
    • MIT License
    • Mozilla Public License

Peranan Software Pada Kehidupan Manusia

  1. Menggantikan Peran Manusia : dengan otomatis melakukan suatu proses atau pekerjaan.
  2. Memperkuat Peran Manusia : dengan membantu manusia untuk menyelesaikan pekerjaannya dengan lebih baik dan tertata.
  3. Restukturisasi Peran Manusia : dengan melakukan perubahan - perubahan thd sekumpulan Pekerjaan atau proses.
  4. Hiburan dan Permainan : Dengan menyajikan aplikasi interaktif hiburan yang semakin dekat dengan kenyataan dan juga tidak membosankan.
